Project Name
IoT Weather Station using ESP32

Project Scope
This project uses ESP32, DHT22, BMP180, SSD1306 OLED Display, and an RGB LED to develop an IoT-based weather monitoring system. The system continuously measures temperature, humidity, atmospheric pressure, and altitude in real time. Based on the pressure values, it predicts weather conditions such as Sunny, Cloudy, or Rainy and indicates the result using an RGB LED. The measured sensor values and weather prediction are displayed on the OLED screen and can also be monitored through the Serial Monitor, making it a compact and efficient weather monitoring solution.
